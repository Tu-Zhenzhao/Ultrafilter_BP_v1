未来发展计划 // FUTURE PLAN

未来发展计划：成为Agent 时代的私有搜索基础设施

AI 会从聊天走向行动；当每个企业和个人都拥有自己的 Agent 系统，最关键的基础设施会是： 让 Agent 稳定调用自己的私有信息

HUMAN-UI LEGACY
2003 – 2023

上一代软件时代

互联网 / SaaS / App 时代

数据库传统程序存取
网页 / APP人类阅读、点击和触屏
SaaS 后台人工管理流程
文件夹 / 知识库人类整理和查找
权限流程人类组织结构

今天很多 Agent 的“神奇操作”，本质是在让 AI 模拟人类使用旧界面

现在 // AI 转折期
聊天 行动 Agent 系统
市场转向 AI 能力正在被浪费在适配旧系统上

它能做到，但这不是最适合 AI 的交互方式，AI 更适合通过结构化接口、命令、权限和数据调用来完成任务

重要提示 ULTRAFILTER 现在 先占住每次 Agent 行动之前最不可绕开的调用层：私有搜索

我们不重建整个 AI OS，我们把企业和个人的私有数据，变成 Agent 可稳定调用的搜索层

NEXT 5–10 YEARS

Agent-native 基础设施时代

AI Systems Inside Work

动作接口人负责审查和授权，Agent 通过命令、API 和工具协议执行任务
数据分发数据从“躺在应用里”，变成可按权限、任务和状态被 Agent 调用的信息流
身份与权限企业会管理 Agent 身份：它能看什么、能做什么、替谁执行、谁来确认
Agent 协作多个子 Agent 分工抓取、分析、执行和审计，主 Agent 或人类做关键决策
精准私有搜索Agent 每次行动前，都需要找到正确、最新、被授权的信息

Ultrafilter 先占住其中最确定的一层：Agent 调用私有数据前的精准搜索层

01 聊天走向行动

AI 从回答工具进入执行系统

02 企业拥有自己的 AI 系统

小团队要助手，大企业要多 Agent 协作

03 旧设施仍为人设计

数据库、网页、APP、权限流程都在适配上一代交互

04 新设施同时服务人和 Agent

人需要界面，Agent 需要可调用的数据和动作接口

05 搜索一定是刚需

AI 不可能读完所有数据，必须检索正确的信息
