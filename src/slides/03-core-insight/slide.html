关键定义

AGENT-READY DATA

进入 Agent 工作流的数据，需要同时满足四个条件：
找得到、用得住、保持新、动得了。

找得到

统一索引、metadata 和检索层，Agent 知道去哪里找。

用得住

权限继承、来源引用与审计记录，结果可被验证。

保持新

支持增量同步与 CRUD，避免过期结果进入流程。

动得了

数据进入工具调用和业务动作，从“读”延伸到“做”。
